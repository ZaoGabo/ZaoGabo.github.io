Coloca aquí tus imágenes para el juego "Encuentra las Diferencias".

Nombres esperados:
- original.png  -> imagen base
- modified.png  -> imagen con diferencias

Sugerencia: optimiza las imágenes a WebP o PNG comprimido para mejorar tiempos de carga.

Ejemplo de uso en `src/App.jsx` (ya configurado):
- `/images/original.png`
- `/images/modified.png`

Después de copiar las imágenes reinicia el servidor de desarrollo si es necesario:

```powershell
# desde la raíz del proyecto
npm run dev
```
