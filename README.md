# Encuentra las Diferencias (prototipo)

Proyecto React + Vite minimal para ejecutar el componente "Encuentra las Diferencias".

Requisitos:
- Node.js (16+ recomendado) y npm

Pasos para ejecutar:

```bash
# desde PowerShell o CMD en Windows
cd "c:\Users\ZaoGabo\Pictures\proyecto"
npm install
npm run dev
```

Luego abre la URL que muestre Vite (por defecto http://localhost:5173).

Notas:
- Usé el CDN de Tailwind para evitar una configuración completa de Tailwind; es suficiente para un prototipo.
- Si quieres integrar Tailwind de manera "correcta" (postcss + config), puedo añadirlo.
- Las imágenes vienen de Imgur; si no cargan reemplaza las URLs en `src/App.jsx`.
